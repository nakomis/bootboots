# ATTRIBUTIONS

This file lists all third-party dependencies used in the BootBoots ESP32-CAM firmware project.

## Embedded Firmware Dependencies

### ESP32 Arduino Core
- **License**: LGPL-2.1 and GPL-2.0
- **Source**: https://github.com/espressif/arduino-esp32
- **Description**: Core Arduino functionality for ESP32 microcontrollers

### ESP32 Camera Driver
- **Package**: espressif/esp32-camera
- **Version**: 2.0.4
- **License**: Apache-2.0
- **Source**: https://github.com/espressif/esp32-camera
- **Description**: ESP32 camera driver for OV2640, OV3660, OV5640, and other camera modules

### ArduinoJson
- **Package**: bblanchon/ArduinoJson
- **Version**: 6.21.3
- **License**: MIT
- **Source**: https://github.com/bblanchon/ArduinoJson
- **Description**: JSON library for Arduino and embedded C++

### ESP32 BLE Arduino
- **Package**: ESP32 BLE Arduino
- **Version**: 2.0.0
- **License**: Apache-2.0
- **Source**: https://github.com/nkolban/ESP32_BLE_Arduino
- **Description**: Bluetooth Low Energy (BLE) library for ESP32

### Arduino MQTT Client
- **Package**: arduino-libraries/ArduinoMqttClient
- **Version**: 0.1.8
- **License**: MIT
- **Source**: https://github.com/arduino-libraries/ArduinoMqttClient
- **Description**: MQTT client library for Arduino

### ArduinoOTA
- **Package**: ArduinoOTA
- **License**: LGPL-2.1
- **Source**: https://github.com/esp8266/Arduino/tree/master/libraries/ArduinoOTA
- **Description**: Over-the-Air update functionality for ESP32

### ESPmDNS
- **Package**: ESPmDNS
- **License**: LGPL-2.1
- **Source**: https://github.com/espressif/arduino-esp32/tree/master/libraries/ESPmDNS
- **Description**: mDNS library for ESP32

### Wire
- **Package**: Wire
- **License**: LGPL-2.1
- **Source**: https://github.com/espressif/arduino-esp32/tree/master/libraries/Wire
- **Description**: I2C communication library for ESP32

### SD_MMC
- **Package**: SD_MMC
- **License**: LGPL-2.1
- **Source**: https://github.com/espressif/arduino-esp32/tree/master/libraries/SD_MMC
- **Description**: SD card interface using MMC protocol for ESP32
