# Camera Settings Reference

## Baseline — Night (with LED strips)

Good starting point for low-light indoor captures with LED strip illumination.

### Resolution & Quality

| Setting | Value |
|---|---|
| Frame Size | UXGA (1600×1200) |
| JPEG Quality | 10 |
| Frame Buffers | 2 |

### Image Quality

| Setting | Value |
|---|---|
| Brightness | 0 |
| Contrast | 1 |
| Saturation | 0 |
| Special Effect | None |

### White Balance

| Setting | Value |
|---|---|
| White Balance | On |
| AWB Gain | On |
| WB Mode | Auto |

### Exposure

| Setting | Value |
|---|---|
| Auto Exposure | Off |
| AEC DSP | On |
| AE Level | 2 |
| AEC Value | 1200 |

> Auto Exposure is disabled and AEC Value set to maximum (1200) to get the longest possible
> exposure in a dark scene. AEC DSP (night mode) is kept on for additional DSP processing.

### Gain

| Setting | Value |
|---|---|
| Auto Gain | On |
| AGC Gain | 5 |
| Gain Ceiling | 6 |

### Corrections

| Setting | Value |
|---|---|
| Bad Pixel Correction | On |
| White Pixel Correction | On |
| Gamma Correction | On |
| Lens Correction | On |
| Downsize Enable | On |

### Orientation

| Setting | Value |
|---|---|
| Horizontal Mirror | Off |
| Vertical Flip | On |

### Test

| Setting | Value |
|---|---|
| Color Bar | Off |

### Flash

| Setting | Value |
|---|---|
| LED Delay (millis) | 1000 |

> 1000ms gives the AEC time to converge after the LED stabilises before the frame is captured.
